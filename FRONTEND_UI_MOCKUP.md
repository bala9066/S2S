## ğŸ¨ Design System

### Color Palette:

```
Primary:   #3B82F6 (Blue)     - Main actions, links
Secondary: #10B981 (Green)    - Success states, approved
Warning:   #F59E0B (Amber)    - Warnings, pending
Danger:    #EF4444 (Red)      - Errors, rejected
Neutral:   #6B7280 (Gray)     - Text, borders

Background:
- Light: #F9FAFB
- White: #FFFFFF
- Dark: #111827
```

### Typography:

```
Headings: Inter (700 - Bold)
Body:     Inter (400 - Regular)
Code:     JetBrains Mono (400)

Sizes:
- H1: 36px (2.25rem)
- H2: 30px (1.875rem)
- H3: 24px (1.5rem)
- Body: 16px (1rem)
- Small: 14px (0.875rem)
```

### Components:

- **Buttons**: Rounded corners (6px), shadows on hover
- **Cards**: White background, subtle shadow, 8px rounded
- **Inputs**: Border on focus, inline validation
- **Tables**: Zebra striping, hover highlight
- **Charts**: Recharts library (responsive SVG)
- **Diagrams**: React Flow library (interactive)
